W06_exclude_wildcard

no www. in exclude

Ensure the web server amber ini file has the following:

amber_excluded_sites[] = wordpress.com

Then browse to this page and check Amber Admin if the following link is cached: https://en.blog.wordpress.com/2015/03/26/saga-satellite/. It shouldn't be because of the wildcard exclusion of all subodmains.

W06_exclude_regex

Ensure the web server amber ini file has the following:

amber_excluded_sites[] = api\.jquery\.com/jQuery\.

Then browse to this page and check Amber Admin if the following links are cached:

  • http://api.jquery.com/jQuery.ajax/
  • http://api.jquery.com/addClass/

The first link shouldn't be cached because of the regex exclusion. The second link should be cached because it doesn't match the regex.

Links tested with Perl Regex Tester.
